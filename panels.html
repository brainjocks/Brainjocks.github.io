Style Box Base Components

A style box creates a region of the screen where the user can select background image or color and text color.

  •                   
                  <div class="score-style-box"
                       data-ux-module="score_bootstrap/Components/StyleBox"
                       data-ux-state="loaded" style=""></div>

Modal Base Components

Used to create a popup panel that can display various components

  •                   
                  <div aria-hidden="false" class="modal fade score-modal in" role="dialog" style="">
                    <div class="modal-backdrop fade in" style=""></div>
                    <div class="modal-dialog (modal size class)">
                      <div class="modal-content">
                        <div class="modal-header" id="">
                          <!--Header Text-->
                          <button type="button" class="close"
                                  data-dismiss="modal" aria-label="Close">
                            <span aria-hidden="true">&times;</span>
                          </button>
                        </div>
                        <div class="modal-body">
                          <!--Body-->
                        </div>
                        <div class="modal-footer score-call-to-action">
                          <!--Call to action-->
                        </div>
                      </div>
                    </div>
                  </div>
