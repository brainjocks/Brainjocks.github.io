Buttons

Button Base Components

A Button component is used for placing a single button on the page. Can be inserted into any structural component.

Button toolbar is used for placing toolbars to the page consisting of button-based components such as: Button, Button Group, Button Menu, Split Button Menu, Image Button.

Button group is used for placing a group of buttons on the page.

Button Menu is used for placing button menu on the page. A button menu has popup (or under) menu options that are shown when the button is clicked. The button itself does not link to any other location.

An Image Button is an image that is wrapped with a link.

A Split Button Menu is a split item - the first 1/2 is a button renderings, the second part is a pull down (or up) menu represented by an up or down arrow.
